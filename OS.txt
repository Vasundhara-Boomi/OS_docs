Operating System lies in the category of system software. 
It basically manages all the resources of the computer.

It is designed in such a way that it can manage the overall resources and operations of the computer. 
OS is a fully integrated set of specialized programs that handle all the operations of the computer. 
It controls and monitors the execution of all other programs that reside in the computer.
It also includes application programs and other system software of the computer. 
Examples of Operating Systems are Windows, Linux, Mac OS, etc.
